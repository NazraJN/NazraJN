## 📈 Some fun projects I've done...

- **AI Agents Hackathon:** Developed a supplier negotiation agent for SMEs in Kenya using n8n.  
  **Problem:** SMEs struggle to find reliable suppliers and competitive prices on platforms like Alibaba and AliExpress.  
  **Solution:** Our agent automatically:
  1. Identifies suppliers and their product offerings  
  2. Compares prices against other suppliers  
  3. Selects the supplier matching the SME’s preferences  
  4. Negotiates with the supplier on behalf of the SME (with a human-in-the-loop approval step)  
  5. Prompts the SME to make a payment  
  **Impact:** Simplified sourcing, reduced manual negotiation effort, and helped SMEs find reliable suppliers efficiently.  

- **Market Digest Bot:** Developed an automated bot using n8n that fetches fund fact sheets from fund managers, extracts key market insights on money market performance, and sends summarized updates via Telegram
